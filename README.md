# 🚀 Terraform AWS End-to-End Infrastructure

## 📌 Project Overview
This project demonstrates Infrastructure as Code (IaC) using Terraform to provision AWS resources.

## 🏗️ Architecture
- VPC
- Public Subnet
- EC2 Instance
- Security Group
- S3 Bucket

## 🎯 Objective
- Automate infrastructure provisioning
- Implement real-world AWS setup
- Learn Terraform workflow

## ⚙️ Steps Performed
1. Configured AWS provider
2. Created VPC and networking components
3. Provisioned EC2 instance
4. Created S3 bucket
5. Configured remote backend (S3)

## 🛠️ Commands Used
terraform init
terraform plan
terraform apply

## ✅ Outcome
- Successfully deployed AWS infrastructure
- Learned Terraform modules and state management

## 📘 Learning
- Infrastructure as Code (IaC)
- AWS resource creation using Terraform
- State management
